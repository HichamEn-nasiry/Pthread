Fonctions pthreads en C

int pthread_create(pthread_t *thread, const pthread_attr_t *attr, void *(*start_routine)(void *), void *arg)

Cette fonction crée un nouveau thread. Elle prend comme arguments un pointeur vers un identificateur de thread (pthread_t *), des attributs du thread (pthread_attr_t *), une fonction qui sera exécutée par le thread (void *(*start_routine)(void *)), et un argument optionnel qui sera passé à la fonction du thread (void *arg).

int pthread_join(pthread_t thread, void **retval)

Cette fonction suspend l'exécution du thread appelant jusqu'à ce que le thread spécifié par thread se termine. Elle renvoie la valeur de sortie du thread dans retval, qui peut être récupérée par le thread appelant.

void pthread_exit(void *retval)

Cette fonction termine l'exécution du thread en cours et renvoie une valeur de sortie (retval) facultative. La valeur de sortie peut être récupérée par d'autres threads en utilisant la fonction pthread_join.

int pthread_cancel(pthread_t thread)

Cette fonction envoie une demande d'annulation du thread spécifié par thread. L'annulation n'est pas garantie et dépend de l'état d'annulabilité du thread. Si l'annulation est effectuée, le thread se termine immédiatement.

int pthread_mutex_init(pthread_mutex_t *mutex, const pthread_mutexattr_t *attr)

Cette fonction initialise un verrou mutex spécifié par mutex. Elle peut également prendre un attribut de verrou mutex optionnel (pthread_mutexattr_t *) pour personnaliser le comportement du verrou.

int pthread_cond_init(pthread_cond_t *cond, const pthread_condattr_t *attr)

Cette fonction initialise une variable de condition spécifiée par cond. Elle peut également prendre un attribut de variable de condition optionnel (pthread_condattr_t *) pour personnaliser le comportement de la variable.

int pthread_mutex_destroy(pthread_mutex_t *mutex)

Cette fonction détruit un verrou mutex spécifié par mutex. Elle libère les ressources associées au verrou après son utilisation. Il est important de détruire les verrous inutilisés pour éviter les fuites de mémoire.

int pthread_cond_destroy(pthread_cond_t *cond)

Cette fonction détruit une variable de condition spécifiée par cond. Elle libère les ressources associées à la variable après son utilisation. Il est important de détruire les variables de condition inutilisées pour éviter les fuites de mémoire.

int pthread_barrier_destroy(pthread_barrier_t *barrier)

Cette fonction détruit une barrière de synchronisation spécifiée par barrier. Elle libère les ressources associées à la barrière après son utilisation. Il est important de détruire les barrières inutilisées pour éviter les fuites de mémoire.
